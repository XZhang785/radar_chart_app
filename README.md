# 雷达图分析工具

[English](README_EN.md) | 中文

---

一个基于 Python FastAPI 的雷达图可视化工具，支持多种图表库（ECharts、Chart.js、Plotly），可以自动计算每列平均值并生成对比雷达图。

## 功能特性

### 核心功能

| 特性 | 描述 |
|------|------|
| 📁 **多格式支持** | 支持 CSV、XLSX、XLS 格式文件上传 |
| 📊 **自动统计** | 自动计算每列的平均值 |
| 🕸️ **雷达图对比** | 每个数据行与平均值对比展示 |
| 📏 **归一化处理** | 数据自动归一化到 0-1 比例 |
| 💡 **交互提示** | 悬停显示具体数值和百分比 |
| 📄 **数据表格** | 详细的对比数据表格 |
| 📱 **响应式设计** | 支持桌面和移动设备 |
| ⚡ **分页导航** | 大数据量时支持分页浏览 |

### 导出功能

| 功能 | 描述 |
|------|------|
| 🖼️ **导出单张图** | 导出单个雷达图为 PNG 图片 |
| 📋 **导出长图** | 将所有雷达图合并为一张长图 |
| 📚 **导出多张图** | 批量导出所有雷达图为独立图片 |

### Chart.js 版本特色

| 特性 | 描述 |
|------|------|
| 👤 **单人对比** | 选择单个学生，查看其雷达图与平均值详细对比 |
| 📉 **指标差距** | 直观展示每个指标与平均值的差值（+绿色/-红色） |
| 📊 **详细表格** | 显示实际值、平均值、差值和状态 |
| 🎨 **现代UI** | 使用 Inter 字体，渐变色设计，Toast 提示 |

## 快速开始

### 环境要求

- Python 3.8+
- pip 包管理器

### 安装步骤

```bash
# 1. 克隆项目（如果使用 git）
git clone <repository-url>
cd radar_chart_app

# 2. 创建虚拟环境（推荐）
```

| 版本 | 图表库 | 端口 | 启动命令 |

| **ECharts 版本** | ECharts | 8000 | `python app.py` |

| **Plotly 版本** | Plotly | 8003 | `python app-plotly.py` |

## 数据格式要求

### 输入文件格式

| 列 | 类型 | 说明 |
|----|------|------|
| 第一列 | 字符串 | 行标签（如：姓名、产品名、城市等） |
| 第二列及以后 | 数值 | 用于雷达图分析的数值数据 |

### 示例数据

```csv
姓名,数学,语文,英语,物理
张三,85,90,88,92
李四,72,80,75,78
王五,95,85,92,88
赵六,68,75,70,72
孙七,88,92,85,90
```

### 数据验证规则

- ✅ 至少需要 2 行数据
- ✅ 至少需要 1 列数值数据
- ✅ 支持中文列名
- ✅ 支持 Windows (GBK) 和 UTF-8 编码的 CSV

## 使用指南（Chart.js 版本）

### 1. 上传数据文件

- 点击"选择文件"按钮选择文件
- 或直接拖拽文件到上传区域
- 支持格式: `.csv`, `.xlsx`, `.xls`

### 2. 查看数据预览

上传成功后会自动显示：
- 统计卡片（总行数、维度数、平均值数量）
- 数据预览表格
- 每行的"查看雷达图"按钮

### 3. 单人对比分析

在"单人对比"标签页中：
- 从下拉菜单选择要查看的学生
- 查看该学生的雷达图与平均值对比
- 查看每个指标的详细数据（实际值、平均值、差值、状态）

### 4. 雷达图分析

在"雷达图分析"标签页中：
- 查看所有学生的雷达图分页展示
- 支持导出功能：
  - **导出单张图**: 选择单个雷达图导出
  - **导出长图**: 所有雷达图合并为一张长图
  - **导出多张图**: 批量导出所有雷达图

### 5. 详细数据

在"详细数据"标签页中：
- 查看所有学生的完整数据对比
- 颜色标识高于/低于平均值

### 6. 交互操作

- **悬停**: 鼠标悬停在雷达图上显示详细数值
- **分页**: 使用"上一页"、"下一页"按钮浏览
- **跳转**: 使用下拉框快速跳转到指定页面

## 数据处理流程

```
1. 文件上传
   └── 读取 CSV/Excel 文件
   
2. 数据验证
   └── 检查行列数、数据类型
   
3. 数据计算
   ├── 计算每列平均值
   ├── 找出每列最大/最小值
   └── 归一化处理（0-1 比例）
   
4. 雷达图生成
   ├── 前端渲染图表
   └── 每行数据 + 平均值对比
```

## 雷达图说明

### 归一化公式

```
normalized_value = (original_value - min) / (max - min)
```

- 每个维度的最小值映射为 0
- 每个维度的最大值映射为 1
- 平均值根据其在范围内的位置显示

### 图表元素

| 元素 | 颜色 | 说明 |
|------|------|------|
| 实线区域 | 紫色 (#667eea) | 实际数据 |
| 虚线区域 | 红色 (#ff6b6b) | 平均值 |
| 标注点 | 圆点/方块 | 数据点位置 |

## 常见问题

### Q: 上传文件失败？

**A:** 请确保：
- 文件格式为 CSV、XLSX 或 XLS
- 文件编码为 UTF-8 或 GBK
- 数据包含至少 2 行、1 列数值

### Q: 雷达图显示不完整？

**A:** 
- 刷新页面重新上传
- 检查浏览器控制台是否有错误
- 尝试缩小窗口或调整每页显示数量

### Q: 如何处理中文乱码？

**A:** 
- CSV 文件确保使用 UTF-8 编码
- Excel 文件没有编码问题

### Q: 导出图片失真？

**A:** 
- 使用 Chart.js 版本（推荐）
- 确保图表完全渲染后再导出
- 检查浏览器控制台是否有错误信息
