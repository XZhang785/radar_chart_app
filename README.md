# 雷达图分析工具

一个基于 Python FastAPI + ECharts 的雷达图可视化工具，可以自动计算每列平均值并生成对比雷达图。

## 功能特性

- 📁 支持上传 CSV、XLSX、XLS 格式的文件
- 📊 自动计算每列平均值
- 🕸️ 雷达图展示每个数据行与平均值的对比
- 📏 数据归一化处理（0-1比例）
- 💡 悬停显示具体数值
- 📄 详细的对比数据表格
- 📱 响应式设计，支持分页
- ⚡ 实时交互，无需刷新页面

## 数据格式要求

| 要求 | 说明 |
|------|------|
| 第一列 | 行标签（如：姓名、产品名等） |
| 其余列 | 数值型数据 |
| 最小行数 | 至少2行数据 |
| 最小列数 | 至少1列数值 |

## 示例数据格式

```csv
姓名,数学,语文,英语
张三,85,90,88
李四,72,80,75
王五,95,85,92
```

## 快速开始

### 1. 安装依赖

```bash
cd radar_chart_app
pip install -r requirements.txt
```

### 2. 启动应用

```bash
python app.py
```

### 3. 访问应用

打开浏览器访问: http://127.0.0.1:8000

## 使用方法

1. **上传文件**: 点击"选择文件"按钮或拖拽文件到上传区域
2. **查看预览**: 上传后会自动显示数据预览和统计信息
3. **生成雷达图**: 点击"生成雷达图"按钮
4. **分页浏览**: 使用分页控件浏览不同的雷达图
5. **交互查看**: 鼠标悬停在雷达图上查看具体数值
